# RTStest

Стартовый каркас RTS-проекта на Rust по плану из `RTS_GAME_PLAN.md`.

## Что уже реализовано
- Базовая модель карты (тайлы, проверка проходимости/валидности строительства).
- Экономический контур MVP (добыча золота и дерева).
- Deterministic simulation через фиксированный тик (`FIXED_TICK_MS = 50`).
- OpenGL-ориентированный слой placeholder-текстур как цветных квадратов.
- Рендер в реальном времени через SDL2 в отдельном запущенном окне.

## Запуск
```bash
cargo run
```
После запуска открывается SDL-окно с realtime-рендером.
Управление: `Esc` или закрытие окна для выхода.

## Тесты
```bash
cargo test
```
